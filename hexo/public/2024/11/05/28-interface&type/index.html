interface & type

Zoella Lv4
  2024-11-05 09:27 2024-11-05 09:27 Created   2025-01-03 00:09:20 2025-01-03 00:09:20 Updated  330 Words  1 Mins  

一、相同点

1、都可以描述对象和函数的类型

但语法不一样，type 使用 = 赋值

2、都允许扩展（extends）

并且 interface 和 type 可以互相扩展

（1）interface 扩展 interface

（2）type 扩展 type

（3）interface 扩展 type

（4）type 扩展 interface

二、不同点

1、type 可以，但 interface 不行

（1）基本类型别名

（2）联合类型

（3）元组

具体定义数组每个位置的类型

（4）使用 typeof 获取实例的类型 对 type 进行赋值

2、interface 可以，但 type 不行

interface 能够声明合并

三、总结

优先用 interface，无法实现再用 type，因为 interface 具有更直观和易于理解的特点，并且能够进行声明合并。
