# Gabrielle-hexo-blog

### install Hexo

```npm install -g hexo-cli```


### install dependencies

```cd hexo```

```npm i```


### run local server

```hexo s``` or ```hexo server```


### create new articles

```hexo new <title>```


### generate static files

```hexo g``` or ```hexo generate```


### deploy the site

```hexo d``` or ```hexo deploy```


### generate & deploy

```hexo g -d```
